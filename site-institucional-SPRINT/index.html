Sobre Nós

Somos uma empresa que buscar melhorar a qualidade de vida das pessoas que necessitam dos medicamentos oncológicos através do devido monitoramento deste, utilizando de meios tecnólogicos e inovadores. Mantendo nossos métodos sobre constante melhoria para sempre podermos entregar o que há de melhor a nossa disposição!

Missão

Diminuir a perda e desperdício de remédios.

Visão

Ajudar o consumidor final desses medicamentos.

Valores

Transparência, foco, seriedade e inovação.

Principais serviços prestados

- Soluções para monitoramento de medicamentos

- Soluções para gestão de estoque

- Soluções para Tratamento de dados

Desenvolvimento de tecnologias

Nosso laboratório estuda novas tecnologias do mercado de Ti para a implementação e aprimoramento de nossos trabalhos para melhor atendimento ao cliente

Disponibilidade em tempo real

Através do nosso banco de dados e auxílio do arduino conseguimos armazenar os dados diretamente no site para a disponibilidade do usúario

Monitoramento de ambientes

Nosso sistema monitora e armazena os dados coletados dentro dos ambientes controlados para análises que posteriormente podem ser feitas
